# Hotel Booking System

This is a simple Hotel Booking System developed using *Python Flask* and *SQLite database*.

## Features

- User Registration
- User Login
- Admin Login
- Add Booking
- View Bookings
- Edit Booking
- Payment Page
- Booking Success Page

## Technologies Used

- Python
- Flask
- HTML
- SQLite
- Bootstrap

## Project Structure

hotel-booking-system
│
├── main.py
├── create_db.py
├── database.db
│
├── index.html
├── dashboard.html
├── add_booking.html
├── login.html
├── register.html
└── payment.html

## How to Run

1. Install Flask


pip install flask


2. Run the application


python main.py


3. Open browser


http://127.0.0.1:5000
